## What is Stryde?

Stryde is a performance-focused Minecraft client for version **1.8.9**, built for competitive PvP players. It combines FPS optimization, 30+ gameplay modules, a fully customizable HUD, and offline account support — all in one package.

---

## Features

- ⚡ **FPS Optimization** — Optimized rendering engine for smoother gameplay and higher framerates
- 🎯 **30+ Modules** — A complete suite of competitive gameplay enhancements
- 🎨 **Custom HUD** — Draggable elements, custom fonts, scaling, and clean visual styles
- 🔄 **Offline Account Support** — Works seamlessly with both offline and online accounts
- 🛡️ **Safe & Stable** — Built with account-friendly integrations
- 🌐 **Privacy Controls** — Optional settings to manage connections and improve stability

## Installation

### Requirements
- Minecraft **1.8.9**
- [OptiFine 1.8.9](https://optifine.net/downloads) *(required)*
- Java 8+

### Steps

1. Download and install **OptiFine 1.8.9** first
2. Download the latest **Stryde JAR** from [Releases](https://github.com/YOUR_USERNAME/YOUR_REPO/releases)
3. Place the JAR in your `.minecraft/mods` folder (Legacy Fabric) or launch directly
4. Launch Minecraft and enjoy

> Windows and Linux launchers are coming soon.
